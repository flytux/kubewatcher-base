NCP Sub Account 생성 API : POST/sub-account
파라미터 필수여부 타입 설명 기본값
active Y boolean Sub account 상태 (true:활성, false:비활성) true
canAPIGatewayAccess Y boolean Access Key를 생성하여 해당 키로 API Gateway에 등록된 API를 사용할 수 있음 false
canConsoleAccess Y boolean 서브 계정이 관리 콘솔에 접근할 수있도록 설정 true
consolePermitIps N array 콘솔 접근 가능한 IP대역 목록 Null
Email N String 이메일 사용자 정보의 이메일 set
Loginid Y String 로그인 아이디 사용자 정보의 ID set
Memo N String 메모 “통합관리”
Name Y String 이름 사용자 정보의 성명 set
needPasswordReset Y boolean 비밀번호 재설정 필요여부, True이면 생성된 서브 계정으로 로그인 시 , 새로운 비밀번호 변경하도록 안내 True
Password Y String 비밀번호 (통합관리 사용자 화면의 비밀번호는 일회성) 사용자 정보의 비밀번호 set
useConsolePermitIp N boolean 콘솔 접근시 IP대역 제한을 사용할 수 있음 false
